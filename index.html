SiteBase2

A minimalistic HTML + CSS + JS base front-end starter boilerplate for simple web site projects.

View a Demo Browse on GitHub

Student Friendly

Designed for web designers and front-end developers, SiteBase is a simple HTML + CSS + JS framework created for students learning how to code a basic front-end framework from scratch by hand, line by line.

Step-by-Step Lessons

Nothing Required

No prior web design experience is required. Designed for first-time front-end coders, SiteBase2 uses no pre-processing, no command line, and no special software. All you need is a text editor and a browser.

View the Docs

Customizable CSS

All styling is intentionally minimal for simple, basic web sites. Core CSS includes base typography, a simple responsive grid, and a default site structure with a mobile-friendly navigation system.

Download Zip

Modular, Scalable, and Extensible

SiteBase2 is designed to be a flexible starter set of HTML, CSS, and JS that provides a simple static site folder structure, boilerplate code, and commonly used basic building blocks to help build bespoke static websites from pretty much scratch.
